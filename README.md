# RAG PDF Chatbot

A beginner-friendly Retrieval-Augmented Generation (RAG) chatbot that answers questions from PDF documents using semantic search and a Large Language Model.

## Features

Load PDF documents

Split text into chunks

Create embeddings using Hugging Face

Store embeddings in a FAISS vector database

Retrieve relevant document chunks

Generate answers using FLAN-T5

## Installation

1. Clone the repository

git clone https://github.com/YOUR_USERNAME/rag-pipeline-langchain.git

cd rag-pipeline-langchain

2. Install dependencies

pip install -r requirements.txt

3. Add your PDF

Place a PDF file named notes.pdf in the project folder.

4. Run the chatbot

python rag.py
